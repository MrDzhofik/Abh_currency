## Бот для отслеживания основных курсов валют и металлов

### Отслеживаемые валюты:
1. Доллар США
2. Евро
3. Фунт стерлингов


### Отслеживаемые металлы:
1. Золото
2. Серебро
3. Платина
4. Палладий


## MVP
В первой версии будет выдаваться список с курсами валют на сегодня
В дальнейшем планируется добавить функцию просмотра курсов валют за определенный период и построение графиков
